# Simulación de Seguridad

Pequeña página estática para ejercicios internos de concienciación sobre ingeniería social.

Archivos:
- `index.html` — Página principal con el mensaje del simulacro.

Cómo usar:
1. Abrir `index.html` en un navegador (doble clic o `file://`), o servir con un servidor estático simple:

```bash
# Desde la carpeta del proyecto
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

Notas:
- El enlace de contacto apunta a WhatsApp con el número provisto en la plantilla.
- Esta página es solo para uso interno; no incluir datos sensibles en producción.
